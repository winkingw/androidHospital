## 4. t_doctor_schedule — 排班（动态生成）

```java
// 逻辑：App 首次启动时，为每位在职医生生成未来 7 天排班
// 上午: 08:00-12:00, 下午: 13:00-17:00
// 上午+下午均出诊

for (Doctor doctor : allDoctors) {
    for (int day = 1; day <= 7; day++) {
        String date = 当前日期 + day 天;
        // 上午排班
        insert DoctorSchedule(doctor.id, date, "MORNING", "08:00:00", "12:00:00");
        // 下午排班（每天出诊）
        insert DoctorSchedule(doctor.id, date, "AFTERNOON", "13:00:00", "17:00:00");
    }
}
// 20 医生 × 7 天 × 2 时段 = 280 条排班
```

## 5. t_register_source — 号源（排班创建时自动生成）

```java
// 每条排班按 20 分钟拆分：
// 上午 08:00-12:00 → 12 段（08:00-08:20, 08:20-08:40, ..., 11:40-12:00）
// 下午 13:00-17:00 → 12 段（13:00-13:20, 13:20-13:40, ..., 16:40-17:00）
// 每段默认 total_num=1, remain_num=1, register_fee=20.00

void generateSlots(long scheduleId, String startTime, String endTime) {
    LocalTime start = LocalTime.parse(startTime);
    LocalTime end = LocalTime.parse(endTime);
    while (start.isBefore(end)) {
        LocalTime slotEnd = start.plusMinutes(20);
        insert RegisterSource(scheduleId, start, slotEnd, 1, 1, 20.00);
        start = slotEnd;
    }
}
// 约 238 排班 × 12 段 ≈ 2856 条号源
```

## 18. 数据初始化时机

```
App 首次启动时（SharedPreferences 标记 isFirstLaunch）：
  DBHelper.onCreate()
    → 执行建表 SQL（16 张表）
    → 调用 MockDataUtil.initAll(dbHelper)

MockDataUtil.initAll():
  1. insertDepartments()           // 12条科室
  2. insertDoctors()               // 20条医生
  3. insertUsers()                 // 3条用户
  4. generateSchedulesAndSources() // 7天排班 + 20分钟号源
  5. insertAppointments()          // 8条预约
  6. insertMedicalCards()          // 2条医保卡（先于缴费单，确保外键可用）
  7. insertPaymentOrders()         // 8条缴费单
  8. insertMedicalCardRecords()    // 8条消费记录
  9. insertVisitHistories()        // 5条就诊历史
  10. insertMessages()             // 9条消息
  11. insertBanners()              // 4条轮播图
  12. insertFeedbacks()            // 5条评价
  13. insertAdminUsers()           // 3条后台账号
  14. insertSymptomRules()         // 24条导诊规则(选做)
  15. insertHelpContent()          // 帮助内容(选做/可硬编码)
```
